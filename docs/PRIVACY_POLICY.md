# Privacy Policy for Site Link Explorer

**Effective Date:** January 2025
**Last Updated:** October 30, 2025

---

## Overview

Site Link Explorer is a Chrome browser extension that helps users discover and categorize publicly accessible links from website sitemaps and robots.txt files. This Privacy Policy explains how the extension handles data.

---

## 1. Data Collection

**Site Link Explorer does NOT collect, store, or transmit any personal information or browsing data to external servers.**

The extension operates entirely locally on your device. We do not:
- ❌ Collect personal information
- ❌ Track your browsing history
- ❌ Monitor your online activity
- ❌ Share data with third parties
- ❌ Use analytics or tracking tools
- ❌ Display advertisements

---

## 2. Local Storage Only

All data generated by the extension is stored **locally on your device** using Chrome's built-in storage API:

### What is stored locally:
- **Scan results:** URLs discovered from sitemap.xml and robots.txt files
- **User preferences:** Theme (light/dark), language selection, auto-scan settings
- **Custom categories:** User-created categories and pattern matching rules
- **Scan history:** Recent scan results for quick access

### Where it's stored:
- All data is stored in Chrome's local storage on your computer
- No data is ever uploaded to external servers
- No data leaves your device

### How to delete:
- Clear scan history: Settings → Scan History → Clear History
- Clear cache: Settings → Clear Cache
- Uninstall extension: Removes all local data automatically

---

## 3. Network Requests

The extension makes network requests **only** to:
- Fetch publicly accessible `sitemap.xml` files from websites you visit
- Fetch publicly accessible `robots.txt` files from websites you visit

These files are:
- ✅ Publicly available (no authentication required)
- ✅ Standard web files designed to be read by automated tools
- ✅ Fetched only from websites you explicitly visit
- ✅ Not sent to any external servers

**No other network communication occurs.**

---

## 4. Permissions Explanation

The extension requests the following permissions for its core functionality:

### `activeTab`
- **Purpose:** To read the URL of the website you're currently viewing
- **Usage:** When you click the extension icon, it needs to know which website to scan
- **Privacy:** Only accesses the active tab when you interact with the extension

### `storage`
- **Purpose:** To save your preferences and scan results locally
- **Usage:** Stores settings (theme, language), custom categories, and scan history
- **Privacy:** All data stored locally on your device using Chrome's storage API

### `tabs`
- **Purpose:** To enable the auto-scan feature and update the badge counter
- **Usage:** Detects when you navigate to a new page (if auto-scan is enabled)
- **Privacy:** Only monitors page load events, not page content

### `host_permissions (<all_urls>)`
- **Purpose:** To fetch sitemap.xml and robots.txt files from any domain you visit
- **Usage:** Makes HTTP requests to download these public files
- **Privacy:** Only fetches publicly accessible files, no access to private data

**All permissions are used exclusively for the extension's stated functionality.**

- ❌ Does NOT use external servers

**The extension is completely self-contained and offline.**

---

## 7. User Control

You have full control over your data:

### Data Management:
- **View scan history:** Settings → Scan History
- **Clear scan history:** Settings → Scan History → Clear History
- **Clear cache:** Settings → Clear Cache button
- **Disable auto-scan:** Settings → General Settings → Auto Scan toggle
- **Export data:** Export button in popup (CSV/JSON format)

### Complete Data Removal:
Uninstalling the extension removes all local data automatically. No data persists after uninstall.
